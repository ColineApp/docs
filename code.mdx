## Coline Code

Coline Code is the development workspace built into Coline. It combines code editing, project navigation, AI assistance, and collaboration tools in a shared workspace.

Use Coline Code to work on application code, scripts, configuration files, documentation, and automation projects without switching between separate tools.

## Editor Features

### Monaco-Based Editor

The editor includes:

- Syntax highlighting for 50+ languages
- Multiple open files and tabs
- Inline search and replace
- Automatic indentation and formatting support
- Keyboard shortcuts similar to VS Code

### Project Navigation

Browse and manage project files with the built-in file explorer:

- Nested folders and file trees
- Quick file switching
- Drag-and-drop organization
- Context actions for creating, renaming, and deleting files

### Multi-File Workflows

Coline Code is designed for full project workflows rather than single-file editing:

- Open and edit multiple files at once
- Navigate between related files quickly
- Keep documentation, configuration, and source code together

## AI-Powered Development

Kairo is integrated directly into the coding experience.

### Inline Assistance

Ask Kairo to:

- Explain code
- Generate boilerplate
- Refactor functions
- Fix errors
- Write tests
- Update documentation

### Workspace Context

Kairo can use context from your Coline workspace while helping with development tasks:

- Reference docs and notes
- Review related tasks
- Summarize technical discussions
- Generate code from product requirements

For sandboxed JavaScript execution and automation workflows, see [Kairo Code Execution](/docs/kairo/code-execution).

## Git & Version Control

Coline Code supports git-based development workflows:

- View changed files
- Review diffs
- Stage and commit changes
- Switch branches
- Work with remote repositories

Version control actions are designed to work alongside collaborative editing and AI-assisted changes.

## Supported Projects

Coline Code works well for:

- TypeScript and JavaScript applications
- Python projects
- Configuration repositories
- Documentation sites
- Scripts and automation tools
- Full-stack web applications

Language tooling and framework support may vary by workspace configuration.

## Collaboration

Because Coline Code is part of the broader Coline workspace, development work stays connected to the rest of your team:

- Link code changes to tasks
- Reference docs and discussions
- Collaborate in shared workspaces
- Use channels for engineering communication
