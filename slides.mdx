## Slides

Slides are presentation decks built from blocks. Each slide is composed of the same block types as docs, enabling seamless conversion between documents and presentations.

## Features

- **Block-based slides** — Same blocks as docs (text, images, embeds, tables)
- **Themes** — Built-in and custom slide themes
- **Layouts** — Smart layouts that adapt to content
- **Present mode** — Full-screen presentation with speaker notes
- **Export** — PDF, images, or shareable links

## From Docs to Slides

Convert any doc to slides automatically. Coline intelligently splits content into slides based on headings and structure.
