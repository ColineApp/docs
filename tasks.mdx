## Tasks

Tasks are flexible visual workspaces for organizing anything. Use them for task tracking, planning, brainstorming, or any workflow that benefits from a spatial layout.

## Features

- **Infinite canvas** — Zoom and pan across your board
- **Cards** — Rich content cards with properties
- **Layouts** — Kanban columns, freeform, or grid layouts
- **Properties** — Custom fields with typed data
- **Filters** — Dynamic filtering by any property
- **Automation** — Trigger actions on card changes

## Use Cases

- Sprint planning
- Content calendars
- Bug tracking
- Roadmapping
- Brainstorming sessions
