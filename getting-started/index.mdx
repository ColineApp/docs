## Getting Started

Welcome to Coline. This guide covers the essentials of working in your workspace.

## Your First Steps

### 1. Understand Your Workspace

Your workspace is your team's home in Coline. Everything lives here: files, channels, messages, and calendar events.

### 2. Create Your First File

Files are the heart of Coline. Everything is a file:

- **Docs** (`.doc`) — Traditional rich text documents, like Word or Google Docs
- **Sheets** (`.sheet`) — Structured data with typed columns, formulas, and views
- **Boards** (`.board`) — Flexible canvas for cards, workflows, and visual organization
- **Notes** (`.note`) — Block-based instant capture with backlinks and daily notes
- **Slides** (`.slide`) — Presentation decks with block-based composition
- **Code** (`.code`) — Code files with syntax highlighting (full IDE coming soon)

Create a file from the command bar (`Cmd/Ctrl+K`) or by clicking **New** in any drive.

### 3. Join a Channel

Channels are collaboration spaces with customizable tabs. Each channel combines messaging with files, folders, and apps in one place. Browse public channels or create private ones for sensitive work.

### 4. Try Kairo

Press `Cmd/Ctrl+K` to open the command bar. Type naturally to search, or ask Kairo questions like:

- "Find my recent docs"
- "Summarize #engineering channel"
- "Schedule a meeting with the design team"
