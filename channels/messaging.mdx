## Messaging

Channel messages support rich formatting, threading, and powerful integrations.

## Message Features

- **Rich text** — Formatting, lists, code blocks
- **Mentions** — @person, #channel, [[file]]
- **Attachments** — Drag and drop files
- **Reactions** — Emoji reactions to messages
- **Threads** — Reply in threads to keep conversations organized
- **Pins** — Pin important messages to the channel

## Voice & Video

Start voice or video calls directly in any channel. Screen sharing and collaborative cursors included.
