## Files in Channels

Files and channels are deeply integrated. Share, preview, and collaborate without switching contexts.

## Sharing Files

- **Paste a link** — Coline automatically unfurls file previews
- **Upload** — Drag and drop directly into messages
- **Create** — Make new files from the attachment menu
- **Reference** — Use [[filename]] to link to files inline

## File Previews

Most file types render inline:
- Docs, Notes — Full preview with scroll
- Sheets — First rows visible
- Boards — Card count and status
- Images & Video — Direct embed
- Code — Syntax highlighted snippets

## Collaboration

- **Comments** — Discuss files without leaving the channel
- **Live cursors** — See who's viewing files in real-time
- **Permissions** — Channel access doesn't grant file access; files respect their own permissions
- **Versions** — File history is preserved and accessible
