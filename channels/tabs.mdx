## Tabs

Tabs transform channels from simple chat rooms into rich collaboration spaces. Each tab is a view into a file, folder, or message thread.

## Tab Types

- **Messages** — Default conversation view
- **Files** — Display any file directly in the channel
- **Folders** — Browse drive folders without leaving the channel
- **Apps** — Custom app views and integrations

## Managing Tabs

- Add tabs from the + menu next to existing tabs
- Drag to reorder tabs
- Rename tabs to match your workflow
- Set default tabs for new channel members

## Use Cases

**Engineering Channel**
- Messages (default)
- Sprint Board (board file)
- Bug Tracker (sheet)
- Runbooks (folder)

**Design Channel**
- Messages
- Design System (doc)
- Brand Assets (folder)
- Feedback (board for async critiques)
