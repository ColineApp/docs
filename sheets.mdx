## Sheets

Sheets are structured data tables with typed columns, formulas, and multiple views. Think database meets spreadsheet: typed data with the flexibility of a grid.

## Features

- **Typed columns** — Text, number, date, boolean, url, select, multi-select
- **Formulas** — Auto-complete formula editor with cell references
- **Views** — Grid, calendar, gallery, and custom view types
- **Relations** — Link records between sheets
- **Permissions** — Row-level access control
- **Import/Export** — CSV, JSON, and API access
